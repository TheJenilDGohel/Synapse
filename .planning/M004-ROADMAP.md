# Roadmap - Milestone 4: Astro Starlight Documentation

## Objective
Reconstruct the Synapse documentation site using Astro Starlight for better performance, static hosting compatibility (Firebase), and alignment with the project's original documentation strategy.

## Phases

- [x] **Phase 1: Astro Starlight Setup** - Initialize the Astro project and apply Synapse branding.
- [x] **Phase 2: Content Migration** - Port documentation content to Starlight/MDX format.
- [x] **Phase 3: Firebase Deployment & CI/CD** - Set up Firebase Hosting and automated deployment workflow.
