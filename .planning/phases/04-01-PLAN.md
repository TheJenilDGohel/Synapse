# 04-01: Astro Starlight Setup

## Objective
Bootstrap the Astro Starlight documentation site in the `docs/` folder and integrate it into the root workspace.

## Implementation Steps
1. **Initialize Astro Starlight**: Run `npm create astro@latest -- --template starlight --yes --install --git` in a temporary directory or directly in `docs/` (but I need to be careful with existing files).
   - Actually, I'll create the `docs/` folder and initialize it manually to ensure precision.
2. **Add Workspace Scripts**: Add `docs:dev`, `docs:build`, and `docs:preview` to the root `package.json`.
3. **Configure Starlight**: Update `astro.config.mjs` with Synapse project details (title, logo, social links).
4. **Custom Styling**: Add initial Synapse branding (Blue/Indigo theme) to `src/content/docs/index.mdx` and global CSS.

## Verification & Testing
- `npm run docs:dev` should start the development server.
- `npm run docs:build` should produce a static build in `docs/dist/`.
