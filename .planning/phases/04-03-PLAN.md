# 04-03: Firebase Deployment & CI/CD

## Objective
Configure Firebase Hosting for the new Astro documentation site and set up automated deployment via GitHub Actions.

## Implementation Steps
1. **Firebase Configuration**: Create `firebase.json` and `.firebaserc` in the project root.
2. **GitHub Action**: Create `.github/workflows/deploy-docs.yml` to build and deploy to Firebase on push to main.
3. **Root Integration**: Add `docs:deploy` script to root `package.json`.

## Verification & Testing
- Workflow should be syntactically valid (checked via `npm run check`).
- Note: Actual deployment requires a `FIREBASE_SERVICE_ACCOUNT_SYNAPSE_DOCS` secret to be set in GitHub.
