# 05-02: Visual Overhaul (Neural Aesthetic)

## Objective
Apply the 'Neural Aesthetic' to the documentation site by implementing the Geist font family and the Obsidian/Teal color palette.

## Implementation Steps
1. **Apply Geist Font**: Ensure `Geist Sans` and `Geist Mono` are the primary font families in `global.css` and `custom.css`.
2. **Obsidian Background**: Set the main background color to `#050505` (bg-obsidian).
3. **Signal Teal Accents**: Update primary colors and active states to `#2dd4bf` (signal-teal).
4. **Minimalist Borders**: Reduce visual noise by thinning borders and using darker zinc shades for secondary elements.
5. **Starlight UI Customization**:
   - Update `custom.css` to override Starlight's default colors (gray-900, etc.) with the Obsidian palette.
   - Refine the sidebar neural glow effect.

## Verification & Testing
- `npm run docs:build` should succeed.
- Visual inspection: The site should have a dark, high-contrast look with teal accents and sharp Geist typography.
