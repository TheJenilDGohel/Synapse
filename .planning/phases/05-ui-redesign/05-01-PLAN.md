# 05-01: Foundation & Stack Integration

## Objective
Establish the technical foundation for the new Synapse UI by integrating Tailwind CSS, React, and shadcn/ui.

## Implementation Steps
1. **Add Tailwind CSS**: Run `npx astro add tailwind` in the `docs/` folder.
2. **Add React**: Run `npx astro add react` in the `docs/` folder.
3. **Install shadcn/ui Dependencies**: Install `lucide-react`, `clsx`, `tailwind-merge`.
4. **Initialize shadcn/ui**: Set up the `components.json` and basic directory structure.
5. **Geist Font Integration**: Add the Geist Sans/Mono fonts to the project.
6. **Verification**: Create a `TestButton` using shadcn/ui and render it in `docs/src/content/docs/index.mdx`.

## Verification & Testing
- `npm run docs:dev` should start without errors.
- The `TestButton` should render with Tailwind styles and React functionality.
