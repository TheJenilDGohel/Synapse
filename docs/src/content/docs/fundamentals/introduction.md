---

Synapse is a local-first MCP server providing a unified context layer for AI agents. 

It combines three powerful pillars:
1. **Semantic Code Search**: Deep understanding of your codebase.
2. **Temporal Knowledge Graph**: A structured memory of facts and relations.
3. **Persistent AI Memory**: Long-term recall across sessions.

## The Problem
AI agents today suffer from "context amnesia." They forget what you told them two sessions ago, they struggle to find the right code in large repositories, and they can't track how facts change over time.

## The Solution
Synapse provides a persistent, queryable, and structured context layer that lives directly on your machine. 

- **Local-First**: All data is stored in a local SQLite database.
- **Agent-Native**: Built specifically for MCP-compliant agents like Cursor, Windsurf, and Gemini.
- **Unified**: One tool for search, memory, and knowledge.
