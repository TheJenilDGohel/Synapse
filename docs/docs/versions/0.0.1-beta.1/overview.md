# 0.0.1-beta.1 Overview

<div className="docPanel docPanel--compact">
  <p>
    This is the first documented Synapse beta. It establishes semantic indexing and hybrid search,
    but it predates later cleanup, naming consistency, and runtime hardening.
  </p>

This is the first published Synapse beta documented in the changelog.

## Major additions

- single-package root layout
- `synapse-doctor`
- first semantic indexing service
- `index_status`
- `index_project`
- `search_hybrid`

## Caution

This version predates later naming cleanup and several retrieval/runtime fixes.
